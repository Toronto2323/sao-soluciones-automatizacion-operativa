# Integración sutil de isotipo SAO

Copia tus archivos SVG en esta ruta del proyecto:

assets/icons/

Obligatorios para esta versión:
- 05_isotipo_sao.svg
- 11_favicon_sao.svg

Los demás íconos pueden permanecer guardados para usos futuros, pero esta versión no los fuerza en todas las tarjetas. Se conserva Lucide para íconos funcionales y se usa el isotipo SAO de forma sutil en:
- favicon del navegador
- logo del header
- marca de agua discreta en paneles principales

Esto evita saturar la interfaz y mantiene la identidad visual de forma elegante.
